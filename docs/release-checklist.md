# Release Checklist

- [ ] `VERSION` 与 `CHANGELOG.md` 一致。
- [ ] 完整单元测试和 Python 编译检查通过。
- [ ] Skill 官方结构验证通过。
- [ ] CI、CodeQL 和依赖审查通过。
- [ ] 未提交 `__pycache__`、真实案件数据、密钥、令牌或审批凭证。
- [ ] 法律规则变化附官方来源、效力类型、地域和时间快照。
- [ ] 新增 P0/P1 对抗问题已有可归责人工处置。
- [ ] README、可靠性契约、威胁模型和安全政策与行为一致。
- [ ] 发布标签使用 `vMAJOR.MINOR.PATCH`，发布说明列出兼容性和残余风险。
